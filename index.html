UBC CPEN 355 (2025-26 Winter Term 2): Machine Learning with Engineering Applications

This course frames machine learning as a principled component of intelligence—beyond mere model training—developing foundational understanding from kernel methods to modern deep learning. So students can both use existing tools effectively and design, analyze, and improve algorithms rather than treating them as black boxes.
